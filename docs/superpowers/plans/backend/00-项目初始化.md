### 2.3 配置管理 (app/config.py)

```python
from pydantic_settings import BaseSettings
from pathlib import Path

class Settings(BaseSettings):
    # App
    APP_NAME: str = "Face Scan API"
    DEBUG: bool = True
    
    # Database
    DATABASE_URL: str = "sqlite:///./data/face_scan.db"
    
    # Face Recognition
    FACE_THRESHOLD: float = 0.6
    LIVENESS_FRAMES: int = 5
    
    # Storage
    DATA_DIR: Path = Path("./data")
    FACES_DIR: Path = DATA_DIR / "faces"
    IMAGES_DIR: Path = FACES_DIR / "images"
    ENCODINGS_DIR: Path = FACES_DIR / "encodings"
    
    class Config:
        env_file = ".env"

settings = Settings()
```

### 2.4 数据库连接 (app/database.py)

```python
from sqlalchemy import create_engine
from sqlalchemy.ext.declarative import declarative_base
from sqlalchemy.orm import sessionmaker
from app.config import settings

engine = create_engine(
    settings.DATABASE_URL,
    connect_args={"check_same_thread": False}  # SQLite only
)

SessionLocal = sessionmaker(autocommit=False, autoflush=False, bind=engine)

Base = declarative_base()

def get_db():
    """Dependency for getting database session"""
    db = SessionLocal()
    try:
        yield db
    finally:
        db.close()
```

### 2.5 FastAPI 入口 (app/main.py)

```python
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware
from app.config import settings

app = FastAPI(
    title=settings.APP_NAME,
    debug=settings.DEBUG
)

# CORS
app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],  # 生产环境需限制
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

@app.get("/")
def root():
    return {"message": "Face Scan API", "version": "1.0.0"}

@app.get("/health")
def health_check():
    return {"status": "healthy"}
```

### 2.6 启动脚本 (run.py)

```python
import uvicorn
from app.config import settings

if __name__ == "__main__":
    uvicorn.run(
        "app.main:app",
        host="0.0.0.0",
        port=8000,
        reload=settings.DEBUG
    )
```

## 三、实现步骤

```bash

```

## 五、注意事项

1. **dlib 安装问题**: 
   - Windows 需要安装 CMake 和 Visual Studio Build Tools
   - 或使用预编译包: `pip install dlib-19.24.2-cp310-win_amd64.whl`

2. **虚拟环境**: 强烈建议使用 venv 隔离依赖

3. **端口冲突**: 如 8000 被占用，修改 run.py 中的 port 参数
