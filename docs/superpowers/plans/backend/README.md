## 四、关键决策点

1. **人脸识别库选择**: 使用 `face_recognition` (基于 dlib)，后续可替换为 ArcFace
2. **活体检测方式**: 眨眼检测 (EAR算法) + 点头检测，无需额外硬件
3. **特征向量存储**: .npy 文件存储，内存中加载所有已知特征进行1:N比对
4. **实时通信**: WebSocket 用于实时视频流，HTTP 用于普通API

## 五、验收标准

- [ ] 所有API接口可正常调用（通过 /docs 测试）
- [ ] 人脸识别准确率 > 85%
- [ ] 活体检测能有效拦截照片攻击
- [ ] 考勤记录可正常导出Excel
- [ ] WebSocket 实时视频流延迟 < 500ms
