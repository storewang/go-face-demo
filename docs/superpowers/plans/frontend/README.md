## 四、关键决策点

1. **构建工具**: 使用 Vite（快速开发体验）
2. **UI框架**: Element Plus（组件丰富、中文友好）
3. **状态管理**: Pinia（Vue 3 官方推荐）
4. **摄像头**: 使用 WebRTC API (getUserMedia)
5. **实时通信**: WebSocket 用于实时视频流

```javascript
// Canvas 转 Base64
const base64 = canvas.toDataURL('image/jpeg', 0.8).split(',')[1]
```

## 八、浏览器兼容性

| 浏览器 | 版本要求 | 说明 |
|--------|---------|------|
| Chrome | >= 80 | 推荐 |
| Firefox | >= 75 | 支持 |
| Edge | >= 80 | 支持 |

**注意**: Safari 对 WebRTC 支持有限，建议使用 Chrome

## 九、安全考虑

1. **HTTPS**: 生产环境必须使用 HTTPS（WebRTC 要求）
2. **权限请求**: 摄像头访问需用户授权
3. **CORS**: 后端需配置允许前端域名
4. **XSS**: 使用 Vue 3 自动转义
5. **敏感数据**: 不在前端存储敏感信息

## 十、性能优化

1. **路由懒加载**: 减少首屏加载时间
2. **图片压缩**: Base64 使用 JPEG 0.8 质量
3. **帧率控制**: WebSocket 发送频率 5-10 fps
4. **组件按需加载**: Element Plus 按需引入
5. **生产构建**: 代码分割 + Tree Shaking
